# Letart
起動していると、pixivSketchの配信の開始をお知らせしたり、録画したりできます。  
Alert and record pixiv Sketch stream.  
現在はmacOSのみ対応しています。.app化したものは機能をいくつかついかしたうちに公開します。  
Only macOS.
